# ops-todo

Tag used to categorize operations and infrastructure tasks in TTA.dev.

## Usage

Add `#ops-todo` to any TODO related to infrastructure, deployment, or CI/CD:

```markdown
- TODO Set up PyPI publishing workflow #ops-todo
  type:: infrastructure
  priority:: medium
```
