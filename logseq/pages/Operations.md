# Operations & Maintenance Scripts

## Overview
This page indexes key operational scripts used for maintaining the TTA codebase and agent ecosystem.

## Agent Management
- `scripts/agent_oversight.py`: Main utility for monitoring and managing agent processes.
- `scripts/install_agent_hooks.sh`: Sets up git hooks for agent workflows.
- `scripts/verify-and-setup-persistence.sh`: Ensures agent memory persistence is correctly configured.

## Codebase Health
- `scripts/scan-codebase-todos.py`: Scans for TODOs and generates reports/metrics.
- `scripts/validate_templates.py`: Validates that code templates adhere to standards.
- `scripts/validate-llm-docstrings.py`: Checks docstrings for LLM readability.

## Security & Integration
- `scripts/test_secrets_integration.py`: Verifies secrets management integration.
