## Quick Commands Reference

```bash
# Create experimental branch (Tier 1)
git checkout -b experimental/my-spike
git push origin experimental/my-spike

# Create feature branch for development (Tier 2)
git checkout development
git checkout -b feat/my-feature
git push origin feat/my-feature
# Create PR to development

# Promote to staging (Tier 3)
git checkout staging
git merge development
git push origin staging
# Or create PR from development to staging

# Deploy to production (Tier 4)
git checkout main
# Create PR from staging to main
# Requires manual approval
```
