# dev-todo

Tag used to categorize development tasks in TTA.dev.

## Usage

Add `#dev-todo` to any TODO related to building or improving TTA.dev:

```markdown
- TODO Implement new primitive feature #dev-todo
  type:: implementation
  priority:: high
```
