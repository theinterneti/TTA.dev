# MCP References

This directory contains comprehensive documentation for the various MCP servers that can be integrated with the framework. Each document provides detailed information on how to install, configure, and use the respective MCP server.

## Available MCP Servers

*   **[Context7](context7.md)**: A powerful tool for semantic search and code navigation.
*   **[Database Toolbox](database-toolbox.md)**: A collection of tools for interacting with databases.
*   **[DevContext](devcontext.md)**: A tool for managing and switching between development contexts.
*   **[E2B](e2b.md)**: A tool for running and managing end-to-end tests.
*   **[Filesystem](filesystem.md)**: A set of tools for interacting with the local filesystem.
*   **[GitHub](github.md)**: A comprehensive suite of tools for interacting with the GitHub API.
*   **[Grafana](grafana.md)**: A tool for visualizing and analyzing metrics.
*   **[Hypertool](hypertool.md)**: A tool for building and deploying serverless applications.
*   **[Jaeger](jaeger.md)**: A tool for distributed tracing.
*   **[NotebookLM](notebooklm.md)**: A tool for working with Jupyter notebooks.
*   **[Playwright](playwright.md)**: A tool for browser automation and testing.
*   **[Prometheus](prometheus.md)**: A tool for monitoring and alerting.
*   **[Sequential Thinking](sequential-thinking.md)**: A tool for breaking down complex tasks into smaller, manageable steps.
*   **[Serena](serena.md)**: A powerful, project-based AI coding assistant.
