# Logseq Knowledge Base

**TTA.dev's knowledge management system built with Logseq.**

## Overview

TTA.dev uses Logseq as its primary knowledge management system for:
- Project documentation and guides
- TODO management and task tracking
- Daily journals and session notes
- Learning materials and flashcards
- Architecture decision records

## Structure

### Core Directories
- **`logseq/pages/`** - Main knowledge base pages
- **`logseq/journals/`** - Daily journal entries
- **`logseq/logseq/`** - Logseq configuration

### Key Namespaces
- **`TTA.dev/*`** - Project documentation
- **`TTA.dev/Guides/*`** - How-to guides and tutorials
- **`TTA.dev/Primitives/*`** - Primitive documentation
- **`TTA.dev/Packages/*`** - Package-specific docs
