# TTA.dev Core Memory Bank

This bank contains core knowledge about TTA.dev patterns, primitives, and conventions.

**Scope:** Shared across all worktree agents
**Persistence:** Permanent
**Sync:** Yes, bidirectional with main repository

## Categories

- `implementation-failures/` - Failed approaches to avoid
- `successful-patterns/` - Proven solutions to reuse
- `architectural-decisions/` - Design choices and rationale

## Writing to This Bank

Only write memories that are:
1. **Generalizable** - Applies beyond a single session
2. **Verified** - Solution has been tested and works
3. **Complete** - Includes context, problem, and solution

## Index

The `index.json` file is auto-generated and provides:
- Quick lookup of memory metadata
- Tag-based filtering
- Last-updated timestamps
