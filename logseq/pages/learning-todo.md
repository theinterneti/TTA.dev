# learning-todo

Tag used to categorize learning and documentation tasks in TTA.dev.

## Usage

Add `#learning-todo` to any TODO related to learning or creating educational content:

```markdown
- TODO Write tutorial for RetryPrimitive #learning-todo
  type:: docs
  priority:: medium
```
